Our Commitment

Hudson Valley Ticks is committed to ensuring digital accessibility for people with disabilities.

Conformance Status

Target Level: WCAG 2.1 Level AA

This website aims to comply with Web Content Accessibility Guidelines (WCAG) 2.1 Level AA standards.

Accessibility Features

  • Keyboard navigation support
  • High contrast color schemes (WCAG AA compliant)
  • Alt text for all images
  • Form labels and instructions
  • Proper heading hierarchy
  • Semantic HTML structure
  • Focus indicators on interactive elements
